What is PyLookyloo?

PyLookyloo is the client API for Lookyloo.

Installation

To install the command-line interface for Lookyloo you can run the following command:

pip install pylookyloo

Or clone the repository, and run poetry install.

Usage

PyLookyloo comes with a CLI you can use directly but you can also use it as a library
